# 自动化边界与风险控制

## 1. 为什么不自动改内容和位置

教学课件中的内容、位置、图文关系、动画顺序往往承载教学逻辑。自动化脚本无法可靠理解每一页的教学意图，随意改动可能造成：

- 物理图像与文字对应关系被破坏。
- 光路图、电路图、受力图位置关系错误。
- 动画讲授节奏被破坏。
- 例题解法步骤错乱。
- 视频对象失效。

因此脚本仅做样式统一，不做内容重构。

---

## 2. 低风险自动化项目

- 字体统一。
- 字号统一。
- 颜色统一。
- 背景统一。
- 表格表头加粗。
- 表格正文取消加粗。
- 重点框填充色统一。
- 低风险独立文本公式样式归一。
- 删除明显装饰效果。
- 生成检查报告。

---

## 3. 中风险项目

- 判断视频页并改黑底。
- 判断标题文本框。
- 判断疑似公式。
- 判断公式候选风险等级。
- 判断重点框。

这些项目可自动处理，但必须保留报告。

---

## 4. 高风险项目

默认不自动执行：

- 自动公式转换。
- 自动拆页。
- 自动移动对象。
- 自动改写公式。
- 自动裁剪图片。
- 自动处理组合对象内部结构。
- 自动修改动画。

---

## 5. 推荐安全流程

```text
备份 → 报告 → 小样本试跑 → 人工复核 → 批量处理 → 再次复核 → 导出 PDF
```

---

## 6. 回滚策略

- 原始文件不覆盖。
- 输出文件保存到新目录。
- 报告记录每个文件和页面问题。
- 发现异常时直接回到原始 PPTX。
