# 使用方法

## 1. 环境要求

- Windows 10 / Windows 11
- Microsoft PowerPoint 桌面版
- PowerShell 5.1 或 PowerShell 7+
- 建议使用 `.pptx` 文件，不建议直接处理 `.ppt` 老格式

---

## 2. 第一次使用

把工具包解压到本地目录，例如：

```powershell
D:\tools\physics-ppt-toolkit
```

打开 PowerShell：

```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
cd D:\tools\physics-ppt-toolkit
```

---

## 3. 先生成检查报告

```powershell
.\tools\Report-PhysicsPptStyle.ps1 `
  -InputPath "D:\课件\原始PPT" `
  -OutputDir "D:\课件\检查报告" `
  -Recurse
```

输出：

```text
D:\课件\检查报告\physics-ppt-style-report.csv
```

建议先查看报告中的问题：

- 小字号
- 非规范字体
- 疑似公式
- 视频页背景
- 表格样式
- 过度装饰效果

---

## 4. 批量规范化

```powershell
.\tools\Normalize-PhysicsPpt.ps1 `
  -InputPath "D:\课件\原始PPT" `
  -OutputDir "D:\课件\规范化输出" `
  -Recurse
```

脚本会将规范化后的 PPTX 保存到输出目录，不覆盖原文件。

---

## 5. 单个文件处理

```powershell
.\tools\Normalize-PhysicsPpt.ps1 `
  -InputPath "D:\课件\18.2电功率.pptx" `
  -OutputDir "D:\课件\规范化输出"
```

---

## 6. 只检查不修改

```powershell
.\tools\Normalize-PhysicsPpt.ps1 `
  -InputPath "D:\课件\原始PPT" `
  -OutputDir "D:\课件\检查报告" `
  -Recurse `
  -ReportOnly
```

---

## 7. 使用 VBA 宏

适用于希望在 PowerPoint 内直接运行的情况。

步骤：

1. 打开 PowerPoint。
2. 打开需要处理的 PPTX。
3. 按 `Alt + F11` 打开 VBA 编辑器。
4. 选择 `File > Import File...`。
5. 导入：
   - `vba\PhysicsPptNormalize.bas`
6. 回到 PowerPoint。
7. 按 `Alt + F8`。
8. 运行：
   - `NormalizeCurrentPresentation`

报告型宏：

- 导入 `vba\PhysicsPptReportOnly.bas`
- 运行 `ReportCurrentPresentationStyle`

---

## 8. 建议检查顺序

运行自动化后，人工重点检查：

1. 公式是否显示正常。
2. 视频页是否仍能播放。
3. 动画顺序是否未受影响。
4. 复杂图示是否未被错误改动。
5. 表格是否仍然排版正常。
6. 黑底视频页文字是否清楚。
7. 普通白底页的重点框是否明显。

---

## 9. 重要建议

不要直接在唯一原件上运行任何自动化脚本。始终保留原始课件。
