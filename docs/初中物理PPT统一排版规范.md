# 初中物理 PPT 统一排版规范方案

版本：v1.0  
适用环境：Windows + Microsoft PowerPoint / Microsoft Office  
适用对象：初中物理课堂教学课件、公开课课件、PDF 导出课件

---

## 1. 总目标

课件首先服务课堂教学，而不是装饰展示。统一规范的核心目标是：

1. 后排学生看得清。
2. 教师讲授路径顺畅。
3. 每页重点明确。
4. 公式、图像、实验现象准确清楚。
5. 课件便于长期维护和批量自动化处理。

---

## 2. 总原则

- 页面简单干净。
- 文字论述简炼。
- 每页只突出一个核心知识点或一个主要教学任务。
- 图片、实验图、视频画面尽量放大。
- 色彩一般不超过三种主色。
- 重点内容使用统一颜色和统一文本框样式。
- 不追求花哨动画和复杂装饰。
- 不使用多种字体混排。
- 不使用过小字号。
- 不把讲义式大段文字直接搬到 PPT 页面上。

---

## 3. 自动化边界

自动化脚本只做样式统一和检查报告。

### 3.1 自动化允许修改

- 字体
- 字号
- 字重
- 字体颜色
- 文本框边框颜色和粗细
- 页面背景色
- 表格表头和表格正文样式
- 重点框颜色
- 阴影、发光、渐变等装饰效果
- 疑似公式的检查报告

### 3.2 自动化禁止修改

- 不修改教学内容。
- 不修改文本内容。
- 不修改对象位置。
- 不修改对象大小。
- 不重排图片、公式、文本框。
- 不改变动画顺序。
- 不裁剪图片。
- 不修改物理图像、光路图、电路图、受力图等科学内容。
- 不自动拆页。
- 不自动解释或改写公式含义。

---

## 4. 页面规格

### 4.1 页面比例

统一使用 **16:9 宽屏比例**。

### 4.2 普通页面背景

普通知识页、公式页、例题页、总结页使用白色背景：

```text
#FFFFFF
```

### 4.3 视频页面背景

视频页、暗背景实验现象页、光路动画页使用黑色背景：

```text
#000000
```

视频页面只保留必要标题、观察任务或一句核心提示。视频或实验画面应尽量铺满屏幕。

---

## 5. 字体规范

### 5.1 中文字体

默认使用：

```text
微软雅黑
```

理由：Windows 与 Office 兼容性最好，学校教室电脑最稳。

### 5.2 英文和数字字体

推荐：

```text
Arial 或 Calibri
```

如果与中文混排不协调，可统一使用微软雅黑。

### 5.3 数学字体

Office 自带公式统一使用：

```text
Cambria Math
```

### 5.4 不建议使用

- 宋体作为正文
- 楷体作为正文
- 艺术字作为正文
- 手写风格字体作为正文
- 同一页混用多种中文字体

---

## 6. 字号规范

以正文 **32 pt** 为基准进行适配。

| 内容类型 | 字号 | 字重 |
|---|---:|---|
| 一级标题 | 44–48 pt | 加粗 |
| 二级标题 | 36–40 pt | 加粗 |
| 小标题 | 34–36 pt | 加粗 |
| 正文 | 32 pt | 不加粗 |
| 辅助说明 | 28–30 pt | 不加粗 |
| 最小文字 | ≥ 24 pt | 不加粗 |
| 表头 | 30–32 pt | 加粗 |
| 表格正文 | 28–30 pt | 不加粗 |
| 行内简单公式 | 32–34 pt | 通常不加粗 |
| 独立公式 | 38 pt | 通常不加粗 |
| 核心公式 | 40–42 pt | 通常不加粗 |
| 表格公式 | 28–32 pt | 通常不加粗 |
| 页脚/来源 | 18–22 pt | 不加粗 |

原则：如果内容放不下，优先拆页，不缩小字号。

---

## 7. 字重规范

- 正文默认不加粗。
- 标题加粗。
- 表头加粗。
- 表格正文不加粗。
- 重点词加粗。
- 重点词可配合红色。
- 不整段加粗。
- 不整页加粗。
- 公式通常不额外加粗，主要通过字号、位置、颜色突出。

---

## 8. 颜色规范

### 8.1 白底页面颜色

| 用途 | 色值 |
|---|---|
| 正文 | `#000000` |
| 辅助正文 | `#222222` |
| 重点红 | `#C00000` |
| 公式蓝 | `#0066CC` 或 `#1F4E79` |
| 实验绿 | `#008000` |
| 淡黄底 | `#FFF2CC` |
| 黄底边框 | `#D6A300` 或 `#C9A227` |
| 浅蓝底 | `#D9EAF7` |
| 浅灰底 | `#F2F2F2` |

### 8.2 黑底视频页面颜色

| 用途 | 色值 |
|---|---|
| 正文 | `#FFFFFF` |
| 重点黄 | `#FFD966` |
| 浅蓝提示 | `#4DA3FF` |
| 红色强调 | `#FF4D4D` |
| 绿色提示 | `#00B050` |

### 8.3 颜色语义

| 颜色 | 语义 |
|---|---|
| 黑色 | 普通正文 |
| 红色 | 关键词、易错点、否定项、重要结论中的核心词 |
| 蓝色 | 公式、变量、物理量、已知数据 |
| 绿色 | 实验条件、操作要求、注意事项 |
| 淡黄底 | 本页最重要结论、必须记忆内容、易错提醒 |
| 浅蓝底 | 公式、方法、步骤 |
| 浅灰底 | 普通辅助说明 |

---

## 9. 重点框规范

### 9.1 重点结论框

用于本页最重要结论、易错提醒、必须记忆内容。

```text
背景：#FFF2CC
文字：#000000
关键词：#C00000，加粗
边框：#D6A300 或 #C9A227
边框粗细：1.5–2.25 pt
```

推荐样式：淡黄底 + 黑字 + 红色关键词。

### 9.2 公式方法框

用于公式、方法步骤、解题流程。

```text
背景：#D9EAF7
文字：#000000 或 #1F4E79
```

如果希望课件更简洁，也可不用浅蓝底，直接使用白底公式区。

### 9.3 使用数量

- 每页最多 1 个主重点框。
- 如确有需要，可增加 1 个次级提示框。
- 不建议一页出现多个彩色文本框。

---

## 10. 公式规范

### 10.1 公式对象

重要公式优先使用 Office 自带公式对象。

```text
数学字体：Cambria Math
```

正式展示公式使用上下分子分母的分式形式，不建议长期使用 `P = W/t` 这类斜杠形式作为主公式展示。

### 10.2 分式形式

正式展示：

```text
P = W / t   →   P = W 在上、t 在下的上下分式
η = W有 / W总   →   η = W有 在上、W总 在下的上下分式
I = U / R   →   I = U 在上、R 在下的上下分式
R = U / I   →   R = U 在上、I 在下的上下分式
```

### 10.3 角标规范

角标统一使用下标。

| 普通写法 | 规范含义 |
|---|---|
| W有 | W 下标 有 |
| W总 | W 下标 总 |
| W额 | W 下标 额 |
| G物 | G 下标 物 |
| G动 | G 下标 动 |
| R1 | R 下标 1 |
| R2 | R 下标 2 |
| U1 | U 下标 1 |
| U2 | U 下标 2 |
| I1 | I 下标 1 |
| I2 | I 下标 2 |
| P1 | P 下标 1 |
| P2 | P 下标 2 |

### 10.4 单位规范

- 物理量符号可用斜体。
- 单位必须用正体。
- 数字与单位之间留适当空格。
- 初中 PPT 中乘号建议使用 `×`。

示例：

```text
10 N
0.5 m
6 J
220 V
5 A
44 Ω
600 W
```

### 10.5 公式排版结构

例题页建议按以下顺序：

```text
已知量
依据公式
代入数据
计算结果
答
```

### 10.6 公式自动化原则

- 只自动转换独立公式文本框。
- 不自动转换嵌在长句中的公式。
- 不自动转换图片中的公式。
- 不自动转换视频截图中的公式。
- 不自动转换组合对象内部复杂公式。
- 对疑似公式生成检查报告。
- 转换公式时保持原对象位置和尺寸。
- 转换前必须备份原文件。

---

## 11. 图片与视频规范

### 11.1 图片

- 教学主体图片尽量放大。
- 实验器材图应能看清读数、方向、连接方式。
- 光路图、电路图、受力图中的箭头、线条、文字必须清晰。
- 图片不应过度压缩。
- 图片周围留白不应过大。

### 11.2 视频

- 视频页使用黑底。
- 视频画面尽量铺满屏幕。
- 页面只保留短标题或观察任务。
- 观察任务建议放在视频上方或下方。
- 观察任务文字使用白色或亮黄。
- 不在视频页放大段文字。

---

## 12. 表格规范

- 表格用于总结规律、比较概念、整理实验数据。
- 表格不应承载过多长句。
- 表头使用微软雅黑、加粗。
- 表格正文使用微软雅黑、不加粗。
- 表头字号 30–32 pt。
- 表格正文字号 28–30 pt。
- 表头可使用浅灰底或淡黄底。
- 表格正文使用白底。
- 边框使用灰色或黑色细线。
- 不使用多种强烈填充色。

---

## 13. 版式规范

- 页面边缘保留安全边距。
- 文字不要贴边。
- 主要内容位于视觉中心区域。
- 普通页面标题放在顶部。
- 标题位置尽量固定。
- 图像与对应文字靠近。
- 公式与对应物理图像靠近。
- 箭头、标注应指向明确。
- 每页只解决一个核心问题。
- 每页最多一个核心结论。
- 每页最多一组主图。
- 每页最多一到两个主公式。

---

## 14. 动画规范

- 动画用于分步呈现，不用于装饰。
- 不使用复杂飞入、旋转、弹跳等干扰性动画。
- 公式推导、例题步骤、实验结论可分步出现。
- 同一类内容使用同一种动画。
- 推荐使用出现、淡入、擦除等简单动画。
- 脚本不修改动画顺序。
- 脚本不删除动画。

---

## 15. 自动化批量处理路线

推荐：

```text
PPT 母版 + VBA / PowerShell COM
```

### 15.1 新课件

- 先使用统一 PPT 母版。
- 按页面类型选择模板。
- 公式优先用 Office 自带公式。
- 重点结论使用淡黄底框。
- 视频页使用黑底模板。

### 15.2 旧课件

- 先备份。
- 用 VBA / PowerShell COM 统一字体、字号、颜色、背景、边框、表格。
- 不改内容和位置。
- 只转换或标记低风险独立公式。
- 输出检查报告。
- 人工复核公式、图片、视频、动画和复杂页面。

---

## 16. 自动化检查清单

脚本应检查：

- 小于 24 pt 的文字。
- 非微软雅黑的中文文本。
- 非 Cambria Math 的公式。
- 过多颜色页面。
- 对比度不足文本。
- 一页文本框数量过多页面。
- 艺术字、阴影、发光、渐变对象。
- 疑似公式文本。
- 视频页是否黑底。
- 普通页是否白底。
- 表格表头是否加粗。
- 正文是否过度加粗。
- 疑似图片中文字过小页面。

## 18. 最终策略

新课件用母版控制，旧课件用 VBA / PowerShell COM 批量规范化。自动化只处理样式，不处理教学内容和版式结构。公式只对低风险独立公式进行处理，其余生成报告并人工复核。
